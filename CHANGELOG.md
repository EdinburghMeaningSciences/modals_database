# Changelog

All notable changes are recorded in this file. 

## [1.1.0] - 2026-04-02

This is the stable version that is kept in Edinburgh DataShare with DOI: [https://doi.org/10.7488/ds/8104](https://doi.org/10.7488/ds/8104)

### Fixed 

- Merged Hebrew "tzarikh" and "tzrikha", which are masculine and femine version of the same modal, as the single modal. We have used “tzarikh” as the unified expression, and added the non-? Rows we used to have for "tzrikha" to "tzarikh". 
- Added ? as the "can_express" value for the blank "can_express" cells in Russian "byl by" and Korean "-to + toy-".
- Thanks to Jeremy Kuhn for letting us know of the errors! 

## [1.0.0] - 2024-01-25

This is the first stable release. [The Linguistic Variation paper](https://doi.org/10.1075/lv.23057.ueg) was reviewed in reference to this version.
